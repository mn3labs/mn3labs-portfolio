Compétences

Développement & Sécurité

Un profil orienté sécurité applicative, avec une base solide en développement web et un intérêt fort pour le pentest.

Développement Web

Stack principale

  • PHP natif (procedural) & PDO
  • MySQL / MariaDB
  • HTML5 / CSS3 (dark UI, responsive)
  • JavaScript (validation client, interactions)
  • Bootstrap 5 pour les interfaces modernes
  • Git & GitHub (workflow basique)

Objectif : écrire des applications claires, maintenables et prêtes à être testées du point de vue sécurité.

Sécurité applicative

Concepts

  • Bonnes pratiques OWASP (XSS, SQLi, CSRF, auth)
  • Gestion sécurisée des mots de passe (hash, salage)
  • Protection CSRF et gestion de tokens
  • Notions de rate limiting et protection contre brute force
  • Journalisation des actions sensibles
  • Gestion de sessions & contrôle d’accès

Mise en pratique à travers le projet SecureAuth et de futurs labs.

Outils & mindset

Outils & approche pentest

  • Environnements de dev locaux (WAMP)
  • Utilisation basique de Linux & terminal
  • Découverte des outils type Burp Suite
  • Lecture de documentation officielle & blogs sécu
  • Apprentissage par projets & expérimentation

Objectif : progresser vers un profil orienté pentest / sécurité offensive, avec une forte base technique web.
